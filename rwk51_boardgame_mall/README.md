# 보드게임 쇼핑몰



* 자바 servlet을 통한 쇼핑몰 만들기 프로젝트
* mvc2버전 사용하기
* JDBC , DBCP, jsp , js,html . boostrap, css, 다음 주소 api 등을 활용한 쇼핑몰 만들기. 

# 프로젝트 소개

보드게임을 구매하는 사이트입니다.

여러 종류의 보드게임을 구매할 수 있습니다. 





# 기능 소개

* 회원가입
* 보드게임 구매
* 장바구니 관리
* 물품 검색
* 관리자 모드(보드게임 관리, 유저 관리)
